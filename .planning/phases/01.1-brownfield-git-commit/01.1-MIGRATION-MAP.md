## Batch Details

### Batch A: structure moves
- Scope: 新建标准归档目录，不做代码路径迁移。
- Verify:
  - `git diff --name-status -M`
  - `go test ./...`
- 本批不做什么：
  - 不改动 `cmd/`、`internal/`、`tests/` 代码逻辑
  - 不执行任何删除

### Batch B: path fixes
- Scope: no-op（当前无 import/path 断裂）。
- Verify:
  - `go test ./...`
  - `git diff --name-status -M`
- 本批不做什么：
  - 不引入功能改动
  - 不调整路由/验证行为

### Batch C: docs mapping
- Scope: 产出结构审计、目标树、迁移映射、计划摘要与提交计划。
- Verify:
  - `git diff --name-status -M`
- 本批不做什么：
  - 不替代 AIDP 语义层
  - 不变更需求边界

### Batch D: archive prep
- Scope: 创建归档清单，状态统一 `archived-pending-delete`，删除门禁固定 `human-approval`。
- Verify:
  - `git diff --name-status -M`
- 本批不做什么：
  - 不删除原始文件
  - 不绕过 checkpoint 决策
