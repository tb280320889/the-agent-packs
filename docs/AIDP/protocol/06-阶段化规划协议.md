# 阶段化规划协议

## 当前阶段化来源
阶段设计主要来源于 `docs/改造计划v1/` 的 M0~M6 结果，但进入执行时统一通过 AIDP 语义层表达。

## 规划规则
- phase 目标必须映射到明确 requirement 或验收条件
- 每个 phase 都应说明输入、输出、验证与退出条件
- 旧阶段编号可保留，但不再作为新的唯一入口

## 对 GSD 的要求
- phase 规划需落到 `ROADMAP.md` 或 phase 级工件
- 当前状态需回写 `STATE.md` 与 AIDP runtime 文档
