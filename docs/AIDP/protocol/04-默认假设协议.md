# 默认假设协议

## 基本规则
- 默认假设必须可追溯
- 默认假设不能伪装成已确认事实
- 影响范围、规则、验收的假设必须进入账本

## 回写要求
- 新增假设：写入 `runtime/01-默认假设账本.md`
- 假设被确认：更新状态并在必要时同步到 `core/`
- 假设失效：标记废弃，并在 `runtime/03-变更摘要.md` 说明

## 建议编号
- `A-01`、`A-02` 递增
