# 41 M3 上下文：外部资料纳入、语义映射与资产落位

## 文档类型
单里程碑所需上下文文档

## 一、外部资料候选类型
- skill 说明
- MCP server / resource / tool 文档
- llm.txt
- 官方最佳实践
- 可复用模板与规则清单

## 二、标准吸收流程
1. 收集候选
2. 做语义去重
3. 判断属于：
   - 系统层
   - 主域知识
   - 横线能力知识
   - workflow package 规则
   - 不纳入
4. 做规范映射
5. 形成正式文档资产或候选 backlog

## 三、落位原则
- 系统层资料进入系统层文档，不进入 package README
- 主域资料进入对应主域知识节点或主域文档
- 横线资料进入 capability 空间
- package 专属资料进入 package 合同、模板、tests 或说明文档

## 四、禁止事项
- 把外部资料原样复制为 Blueprint 正文
- 不做语义映射就新增新的 capability 名称
- 用外部资料绕过既有注册表与命名规则
