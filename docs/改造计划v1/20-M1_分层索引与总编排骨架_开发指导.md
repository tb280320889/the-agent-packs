# 20 M1 分层索引与总编排骨架 开发指导

## 文档类型
开发里程碑开发指导文档

## 作用域
M1 负责把“平铺路由”升级为“总编排 -> 主域编排 -> 横线挂接”的结构设计骨架。

## 本里程碑目标
1. 定义节点分类与可见性
2. 定义 global orchestrator 与 domain orchestrator
3. 定义两段式 route 流程
4. 定义横线 attach-only 规则

## 必须产出
1. 节点分类清单
2. 可见性规则
3. 两段式 route 规则
4. 给 M2 的“注册表字段需求”摘要

## 明确不产出
- package 注册表实现细节
- 外部资料吸收流程

## 完成标准
1. 后续 package 新增时知道应该挂在哪个候选空间
2. 横线包不再被视作第一轮主竞争候选
3. M2 可以直接据此设计注册表
