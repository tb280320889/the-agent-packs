# 12 M0 上下文：冻结面、兼容面与禁止事项

## 文档类型
单里程碑所需上下文文档

## 作用域
定义改造计划 v1 中哪些对象是冻结面，哪些属于可演化面。

## 一、冻结面
以下对象顶层语义默认不动：
- Activation Request
- Routing Result
- Context Bundle
- Artifact
- Handoff Bundle
- Validation Plan
- Validator Result
- Activation Result
- 状态枚举与优先级
- workflow package 基本模板

## 二、兼容面
允许演化的对象：
- Blueprint 节点分类字段
- route 的候选集筛选逻辑
- package 注册表
- package 命名与保留名规则
- 文档吸收治理流程

## 三、禁止事项
- 借“重名治理”名义改顶层 envelope
- 借“总编排设计”名义修改 Phase 1 状态语义
- 在没有注册表之前继续新增裸名 package
