# 40 M3 文档吸收管道与知识资产化 开发指导

## 文档类型
开发里程碑开发指导文档

## 作用域
M3 负责把“文档型 agent”从口头设想，升级为可执行的外部资料吸收管道。

## 本里程碑目标
1. 定义外部资料候选池
2. 定义 skill / MCP / llm.txt / best practice 的吸收流程
3. 定义文档型 agent 的落位职责
4. 定义外部资料如何映射回 domain / capability / package 空间

## 必须产出
1. 资料吸收流程
2. 语义映射规则
3. 资产落位规则
4. 给 M4 的迁移与演练建议

## 完成标准
1. 外部资料不再直接散落入仓
2. 文档型 agent 有固定工作流
3. 外部资料可以被有边界地纳入现有知识图谱
