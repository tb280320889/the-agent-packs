# 11 M0 上下文：角色模型、职责与非目标

## 文档类型
单里程碑所需上下文文档

## 作用域
给 M0 提供角色模型、职责边界与非目标定义。

## 一、角色冻结清单
- 项目内部维护 agent
- 迭代开发子 agent
- 文档型 agent
- 用户
- 用户侧外部 agent

## 二、职责冻结

### 项目内部维护 agent
- 接收问题
- 回答项目相关问题
- 路由内部子 agent
- 维护共享上下文与边界说明

### 迭代开发子 agent
- 负责项目内部重构、迭代开发、升级改造

### 文档型 agent
- 负责 skill / MCP / llm.txt / 外部资料的语义整理、规范映射与文档资产化

### 用户侧外部 agent
- 负责在用户本地理解本项目，并通过 MCP 获取必要能力和上下文
- 负责辅助用户搜集可候选纳入的外部资料

## 三、非目标
- 不把项目内部维护 agent 变成外部世界采集器
- 不把文档型 agent 变成实现层 agent
- 不把用户侧外部 agent 直接当作本仓库正式治理角色
